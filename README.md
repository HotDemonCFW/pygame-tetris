# Pygame-Tetris
Ein in Python und Pygame geschriebenes Tetris-Spiel, das im Rahmen eines gemeinschaftlichen Lernprojekts entstanden ist.

## Über das Projekt
Ziel dieses Projekts ist es, die Grundlagen der Spieleentwicklung zu vertiefen. Wir fokussieren uns dabei auf:
- **Game-Loop-Architektur:** Implementierung der Spiellogik und Echtzeit-Rendering.
- **Datenpersistenz:** Speicherung von Highscores in lokalen JSON-Dateien.
- **Konfigurierbarkeit:** Anpassung des Spiels über externe Konfigurationsdateien.

## Technische Details
- **Sprache:** Python 3.x
- **Framework:** Pygame
- **Speicherformat:** JSON (für Highscores und Konfigurationen)

## Steuerung:
 - ← →        Blöcke nach links & rechts verschieben
 - ↑          Blöcke drehen (im Uhrzeigersinn)
 - ↓          Soft-Drop (schneller fallen)
 - Leertaste  Hard-Drop (sofort fallen lassen)
 - P          Pause
 - R          Neustart (nach Game Over)
